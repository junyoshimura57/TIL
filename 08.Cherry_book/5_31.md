- yieldについて  
メソッド内で`yield`を使うとメソッド呼び出しに紐付けたブロックを実行することができる。  
```
def greeting
  puts 'おはよう'
  yield ##ここにdo~endのブロック文が入る
  puts 'こんばんは’
end

grerting do
  puts 'こんにちは'
end
#=>おはよう
#こんにちは
#こんばんは
```


- procについて  
ブロックをオブジェクト化するためのクラス(何らかの処理を表す)。  
```
##hello_procというProcオブジェクトを作る。
hello_proc = Proc.new do
  'Hello!'
end

##Procオブジェクトを呼び出す
hello_proc.call #=>"Hello!"
```
